# 📋 拍摄前准备清单

> 拍摄前 3 天发送给客户，按分类逐一确认

---

## 🧥 服装准备

- [ ] 确认套餐是否含服装（标准及以上含1-2套）
- [ ] 自备服装建议：纯色系（白/黑/米色最百搭）
- [ ] 避免大LOGO、复杂花纹、荧光色
- [ ] 如果有特定风格，带1-2套参考搭配
- [ ] 鞋子：带一双百搭款（白色帆布鞋/黑色皮鞋）
- [ ] Cosplay：确认角色服装和道具准备情况

---

## 💄 妆造准备

- [ ] 套餐含妆造 → 当天到工作室化妆（预留1h）
- [ ] 套餐不含妆造 → 自己化淡妆即可
- [ ] 拍摄前一天敷面膜，当晚早睡
- [ ] 当天不要用新护肤品（防过敏）
- [ ] 带补妆用品：口红/粉饼/梳子
- [ ] 隐形眼镜/美瞳（如需要）

---

## 👜 随身物品

- [ ] 饮用水（户外拍摄必备）
- [ ] 防晒霜（户外）/ 遮阳伞
- [ ] 驱蚊水（夏季/草丛拍摄必备）
- [ ] 充电宝（手机拍照/导航）
- [ ] 小零食（补充能量）
- [ ] 纸巾/湿巾
- [ ] 根据天气带伞/外套

---

## 💡 其他准备

- [ ] 提前找3-5张喜欢的风格参考图
- [ ] 确认拍摄地点和交通路线
- [ ] 预估到达时间，建议提前15min
- [ ] 拍合照的话，同伴也确认好服装
- [ ] 如果带宠物，确认宠物用品和配合度

---

## 📸 拍摄当天流程

| 时间 | 事项 |
|------|------|
| 约定时间 | 到达拍摄地点 |
| 前15min | 沟通当天拍摄计划 |
| 0-30min | 热身拍摄，找感觉 |
| 后续 | 按计划拍摄各场景 |
| 结束 | 付尾款，确认交付时间 |

---

## ⚠️ 户外拍摄特别提醒

| 条件 | 建议 |
|------|------|
| ☀️ 晴天 | 防晒+补水，最好带伞 |
| 🌤️ 阴天 | 最佳光线，不需要特殊准备 |
| 🌧️ 小雨 | 透明雨伞可以当道具 |
| ⛈️ 暴雨 | 免费改期，不扣定金 |
| 🍂 秋冬 | 保暖外套，注意感冒 |
| 🌸 春夏 | 驱蚊水+防晒必备 |
