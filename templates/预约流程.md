# 📅 预约工作流

> 完整预约流程：从咨询到交付

---

## 流程图

```
客户咨询 ──→ 需求诊断 ──→ 套餐推荐
                            │
                            ▼
                       确认意向
                            │
                            ▼
                       选择拍摄日期
                            │
                            ▼
                     ┌─ 天气评估（调用 sunset-prediction）
                     │   ├── 🟢 优质 → 继续
                     │   ├── 🟡 一般 → 备选方案
                     │   └── 🔴 差 → 建议改期
                     │
                     ▼
                  机位推荐 ←── 结合风格/天气/时间
                     │
                     ▼
                  发送准备清单 + 拍摄须知
                     │
                     ▼
                  收取定金（50%）
                     │
                     ▼
                  锁定档期
                     │
                     ├── 拍摄前3天 → 发送提醒+准备清单
                     ├── 拍摄前1天 → 确认天气+时间
                     │
                     ▼
                   拍摄日
                     │
                     ├── 到场地 → 沟通计划
                     ├── 拍摄（2-4h）
                     └── 付尾款
                     │
                     ▼
                   交付流程
                     ├── 3天 → 原片全部交付
                     ├── 7-14天 → 精修交付
                     └── 修改确认 → 最终交付
```

## 关键节点模板

### 拍摄前3天 — AI 自动发送

```
📸 山夏摄影 · 拍摄提醒

你好呀！跟你确认一下 X月X日 的拍摄安排：

📅 日期：X月X日（周X）
⏰ 时间：XX:00
📍 地点：[拍摄地点]
🌤️ 天气：[当天天气预报]

请查收拍摄准备清单，提前做好准备哦~
有任何问题随时问我！
```

### 拍摄前1天 — AI 确认天气

```
🌤️ 明天拍摄天气确认：

[调用 sunset-prediction 获取预报]

如果天气有变化，我们可以在今晚前商量是否改期。
一切正常的话，明天见！😊
```

### 拍摄完成后 — AI 发送交付说明

```
🎉 今天拍摄辛苦了！

照片拍得非常好，相信出片效果会让大家满意的~

接下来时间安排：
📤 原片交付：X月X日前（3个工作日）
🎨 精修交付：X月X日前（7-14个工作日）

收到原片后你可以选要精修的照片，
有任何问题随时找我~
```

## 排期管理

| 状态 | 说明 |
|------|------|
| 🟢 空闲 | 可接单 |
| 🟡 已预约 | 已收定金，锁定档期 |
| 🔴 已满 | 当日已约满 |
| ⚪ 休息 | 摄影师休息日 |

> ⚠️ 排期信息需要手动维护或接入日历 API
