## 三、字段填写规范

### practical 字段

| 字段 | 填写规则 | 缺省处理 |
|------|----------|----------|
| `photography_experience` | 从Q1提取 | 如果没聊到 → "未获取" |
| `target_time` | 从Q2提取 | 如果没聊到 → "未确定" |
| `location_preference` | 从Q6提取 | 如果没聊到 → "交AI推荐" |
| `photo_usage` | 从Q5提取 | 如果没聊到 → "未说明" |
| `budget` | 从第五步½/Q9提取 | 如果没聊到 → "未说明" |
| `raw_photo_preference` | 从Q3提取 | 如果没聊到 → "未讨论" |
| `mbti` | 从Q4提取 | 如果客户不知道/没聊 → "未提供" |
| `confidence_outfit` | 从Q7提取 | 如果没聊到 → "未讨论" |
| `location_preference` | 从Q6提取 | 如果没聊到 → "交AI推荐" |

### aesthetic_level 字段

| 子字段 | 来源 | 判断方式 |
|--------|------|----------|
| `level` | 综合判断 | 审美等级评定方法 |
| `key_signals` | 对话中摘取 | 最能反映审美判断的原话 |
| `recommendation_strategy` | 自动映射 | 见等级对照表 |

### personality 字段

| 子字段 | 映射规则 |
|--------|----------|
| `level` | 人物素养评定方法 |
| `social_energy` | 内向↔外向：用 ● 位置 + 全角横线填充。例：`内向●——————外向`（偏内向）、`内向————●——外向`（偏外向）、`传统———●———探索`（均衡） |
| `communication_style` | 感性↔理性：同上格式 |
| `risk_tendency` | 谨慎↔冒险：同上 |
| `order_preference` | 随性↔有序：同上 |
| `openness` | 传统↔探索：同上 |

> ⚠️ **轴值格式：** 不是 `←●→`（始终在中间），而是用 `●` + 全角横线 `—` 填充，total 长度建议 8-10 字符。位置越靠左=越偏左端标签，越靠右=越偏右端，中间1/3区域=均衡。

---

> 💡 **使用说明**：
> 1. AI 生成完整画像后，同时保存结构化 JSON（用于投递）和人类可读版（存档）
> 2. 评级不是最终结论，而是**沟通的起点**——拍摄当天可能发现新的面向
> 3. 审美等级L1-L2的客户是大多数，这不代表她们不重要——只是需要更多引导
> 4. 评级基准以山夏使用了一段时间后的经验校准为准
