# 🎨 风格指南 — 穿搭 & 场景建议

> 根据拍摄风格推荐的穿搭、配色和场景匹配

---

## 🌸 日系清新

| 项目 | 建议 |
|------|------|
| **穿搭** | 白色连衣裙、浅色衬衫+百褶裙、牛仔+白T |
| **配色** | 白/米/浅蓝/淡粉/浅绿 |
| **场景** | 白塔公园、茅家埠、满觉陇、龙井、馒头山社区 |
| **道具** | 透明伞、草帽、书本、花束 |
| **姿势** | 自然抓拍、回眸、奔跑、低头笑 |
| **出片技巧** | 大光圈虚化背景，逆光拍摄，过曝一点更清新 |

---

## 🎞️ 复古胶片

| 项目 | 建议 |
|------|------|
| **穿搭** | 复古连衣裙、格子衬衫、阔腿裤、贝雷帽 |
| **配色** | 棕/红/橙/黄/墨绿（暖色调） |
| **场景** | 丝联166、馒头山社区、白塔公园、河坊街老巷 |
| **道具** | 胶片相机、老唱片、复古眼镜 |
| **姿势** | 慵懒、随性、做旧时光感 |
| **出片技巧** | 后期增加颗粒感，降低饱和度，暖色调，暗角处理 |

---

## 🎬 电影感人像

| 项目 | 建议 |
|------|------|
| **穿搭** | 风衣、长裙、西装外套、简单但有质感 |
| **配色** | 莫兰迪色系（灰粉、雾蓝、卡其、咖啡色） |
| **场景** | 浴鹄湾、杨公堤、西湖沿线、天目里、都市街头 |
| **道具** | 透明伞、围巾、咖啡杯 |
| **姿势** | 走路抓拍、看向远方、背影、侧脸 |
| **出片技巧** | 2.35:1 宽画幅裁切，电影调色（青橙色调），善用前景虚化 |

---

## 🏯 古风/汉服

| 项目 | 建议 |
|------|------|
| **穿搭** | 汉服/唐装/宋制/明制（自备或租赁） |
| **配色** | 红/金/墨绿/藏蓝 |
| **场景** | 云栖竹径、浴鹄湾、河坊街、胡雪岩故居、乌龟潭 |
| **道具** | 纸扇、油纸伞、古书、剑、灯笼 |
| **姿势** | 抚琴、执扇、凭栏、漫步园林、撑伞回眸 |
| **出片技巧** | 利用烟雾/逆光制造氛围感，低角度拍出建筑气势 |

---

## 🎭 Cosplay

| 项目 | 建议 |
|------|------|
| **穿搭** | 角色服装（自备），道具需提前沟通 |
| **配色** | 根据角色设定 |
| **场景** | 象山艺术公社、青山湖、棚拍+合成 |
| **道具** | 角色武器/法杖/道具（自备或协商） |
| **姿势** | 参考角色原设动作，带角色参考图 |
| **出片技巧** | 后期特效合成，配合角色世界观调色 |

## 🇰🇷 韩式简约

| 项目 | 建议 |
|------|------|
| **穿搭** | 简约西装、针织衫、高腰裤、尖头鞋 |
| **配色** | 米白/卡其/灰/黑（高级感低饱和） |

## 🌿 森系/自然

| 项目 | 建议 |
|------|------|
| **穿搭** | 棉麻长裙、编织草帽、大地色系 |
| **配色** | 绿/棕/米/白（自然色调） |
| **场景** | 龙井茶山、云栖竹径、乌龟潭、青山湖、湘湖 |
| **道具** | 花环、野餐篮、捕梦网 |
| **姿势** | 在自然中行走、触摸树叶、闭眼感受 |
| **出片技巧** | 利用自然光斑，以绿植为前景，浅景深 |

---

## 😢 情绪风/氛围感人像

| 项目 | 建议 |
|------|------|
| **穿搭** | 深色系、宽松型、有垂坠感的材质 |
| **配色** | 黑/灰/深蓝/酒红（深沉压抑） |
| **场景** | 雨天街头、废弃建筑、旧厂房、阴天海边 |
| **道具** | 透明伞、烟饼、纱巾 |
| **姿势** | 低头、闭眼、背影、手挡镜头、雨中 |
| **出片技巧** | 低对比度，暗调，颗粒感，利用阴影制造情绪 |

---

## ⚫ 黑白人像

| 项目 | 建议 |
|------|------|
| **穿搭** | 纯色为主，纹理质感优先 |
| **配色** | 只有黑白灰（光影即一切） |
| **场景** | 简洁背景、有结构感的建筑、强光环境 |
| **道具** | 无（靠光影说话） |
| **姿势** | 侧脸、剪影、强烈透视 |
| **出片技巧** | 注重光影对比，高反差，后期只做亮度曲线 |

---

## 🏙️ 城市街拍

| 项目 | 建议 |
|------|------|
| **穿搭** | 日常但不随便，简约有设计感 |
| **配色** | 黑白灰基础色+一个亮色点缀 |
| **场景** | 白塔公园、象山艺术公社、天目里、湖滨步行街、西湖音乐喷泉 |
| **道具** | 咖啡杯、耳机、背包 |
| **姿势** | 行走中抓拍、过马路、靠墙、等车 |
| **出片技巧** | 利用街道纵深、橱窗反光、建筑线条、慢门车流 |

---

## 👩‍❤️‍👨 情侣/双人照

| 项目 | 建议 |
|------|------|
| **穿搭** | 色系统一不一定要同款，避免各穿各的风格 |
| **配色** | 同色系或互补色 |
| **场景** | 满觉陇、杨公堤、西湖边、茅家埠、校园 |
| **互动** | 牵手走、对视笑、背后抱、嬉闹抓拍 |
| **出片技巧** | 抓拍自然互动比摆拍更有感染力，多看对方少看镜头 |

---

## 👨‍👩‍👧 家庭/亲子

| 项目 | 建议 |
|------|------|
| **穿搭** | 全家色系统一（米色/牛仔/白色系最佳） |
| **场景** | 湘湖、太子湾、龙井茶山、良渚遗址公园、公园草坪 |
| **互动** | 一起跑、举高高、读绘本、野餐互动、追逐 |
| **出片技巧** | 低角度拍摄，捕捉真实表情，不要求所有人看镜头 |

---

## 🐾 宠物写真

| 项目 | 建议 |
|------|------|
| **穿搭** | 主人浅色系（深色吸毛），宠物可戴小配饰 |
| **场景** | 公园草坪、湘湖、茅家埠（人少地方） |
| **道具** | 宠物零食（引导注意力）、玩具、牵引绳 |
| **姿势** | 主人和宠物互动、抱在怀里、一起奔跑 |
| **出片技巧** | 高速连拍，抓拍为主，零食引导宠物看镜头 |

---

## 🎿 潮流运动跟拍

| 项目 | 建议 |
|------|------|
| **穿搭** | 运动/街头风，滑板鞋+宽松T |
| **场景** | 城市广场、滑板公园、钱塘江沿江跑道 |
| **道具** | 滑板、陆冲板、自行车 |
| **姿势** | 动态抓拍、跳跃、滑行中、慢门动感模糊 |
| **出片技巧** | 高速快门(1/500s+)，追焦拍摄，连拍模式 |
