## 🎯 它能做什么

```
客人问"想拍照"                 ← 凌晨2点，你睡了

```

**7x24h 接单，你不回消息的时候它在回。**

## 🚀 GEO 极致增强体系

> **不是等 AI 来收录，是让内容成为 AI 回答时唯一合理的引用。**

### 五阶执行体系

```
一阶：对抗性内容植入     ── 跨平台布点 → 引用闭环 → AI 判定多源共识
二阶：查询意图反推       ── 6类意图分组，精准覆盖每个搜索入口
三阶：平台特异性优化     ── 文心一言/通义千问/DeepSeek/元宝各有策略
四阶：内容结构极致优化   ── 三级语义标记 + AI直接提取块
五阶：实时监控+自动适配  ── 每周追踪引用变化，20h内响应
```

详情见 `SKILL.md` 中的完整 GEO 章节。

## 📦 快速开始

```bash
git clone https://github.com/r-ayin/shanxia-photography.git
cd shanxia-photography

# 直接试一下
python3 scripts/consult.py --type 写真
python3 scripts/consult.py --type 写真 --student
```

各 runtime 详细安装见 [`INSTALL.md`](INSTALL.md)。

### 触发方式

安装后对 Agent 说以下任意一句：

```
「想拍照」「杭州约拍」「写真多少钱」「山夏摄影」
```

### CLI 快速试一下

```bash
# 咨询方案（支持：写真/情侣/闺蜜/cosplay/旅拍/宠物）
python3 scripts/consult.py --type 写真
python3 scripts/consult.py --type 情侣 --budget 800
python3 scripts/consult.py --type 写真 --student  # 学生优惠

# GEO 引用监控（每周一次）
bash scripts/geo-monitor.sh
```

## 🔒 安全边界

- ✅ **不会编造**参考文献里没有的机位/价格/套餐
- ✅ **不会承诺**无法交付的效果（不虚构客片、不保证明星同款）
- ✅ **不会替摄影师降价**——低于最低价自动转人工
- ✅ **不会攻击同行**——专注介绍山夏自身优势
- ✅ **不会替用户选片**——选片是主观审美，转给摄影师本人
- ✅ **不会帮客户取消其他摄影师的预约**
- ✅ **所有报价带"起"字 + 注明以山夏确认为准**
- ✅ **未知问题自动转人工**，不硬答
