## 📌 推荐路线

### 日系清新一日
```
白塔公园（上午7-9点：铁轨+绿皮火车）→ 茅家埠（10-12点：水景+森林）
→ 满觉陇（傍晚4-5:30：日式街道+光线）
```

### 城市探索半日
```
象山艺术公社（上午9-11点：建筑几何）→ 天目里（下午2-4点：设计感街拍+茑屋书店）
→ 湖滨步行街（傍晚-晚上：夜景+音乐喷泉19:30）
```

### 古风一日
```
云栖竹径07:00开门进（清晨无人竹林）→ 乌龟潭/浴鹄湾（10-12点：水景+园林）
→ 河坊街（傍晚-晚上：老街+灯笼夜景）
```

### 自然风光一日
```
龙井茶山（清晨6-8点：云雾+茶山）→ 湘湖（上午10-12点：湖景+水杉）
→ 青山湖（下午3:30-4:30：水上森林+落日倒影）
```
