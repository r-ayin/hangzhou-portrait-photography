```

  — 选自己最喜欢、最自信的衣服来！
  — 纯色最出片（白/黑/米色都好看）
  — 带一套自己穿着会发光的 outfits✨

  山夏这边不含妆造哦～
  所以你可以：
  • 自己画个美美的妆来（淡妆浓妆都可以，看你喜欢的风格）

  前一天敷个面膜早点睡，皮肤状态好了怎么拍都好看！

🎒 **带上**
  — 水
  — 户外的话带驱蚊水和防晒

💡 **最重要的**
  — 放松心情，拍之前好好休息！
  — 想好喜欢的风格参考图发给我

具体还会根据你的套餐再细化，交给我就好啦～✨
```

## 📊 话术使用原则

| 场景 | 使用策略 |
|------|----------|
| 客户主动问 | 直接对回答复，甜甜的但不假 |
| 对比多家 | 夸山夏的好，不踩别人 |
| 犹豫不决 | 推荐轻简套餐先试试水 |
| 砍价 | 提供工作日优惠等替代方案，不直接降价 |
| 第一次拍 | 先暖一暖，再给建议 |
| 投诉/不满 | 先道歉再解决，不说一堆理由 |
