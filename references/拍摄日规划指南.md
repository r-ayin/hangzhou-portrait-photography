## 一、天气评估系统

### 拍摄指数分级

| 指数 | 评分 | 建议 |
|------|------|------|
| 🔥 **绝佳** | 75-100 | 火烧云+好天气，必出外景 |
| 🌤️ **良好** | 60-74 | 适合户外拍摄 |
| 🌥️ **一般** | 35-59 | 可拍但不出彩，备选室内 |
| ☁️ **较差** | 0-34 | 建议改期或转室内棚拍 |

### 各天气条件的拍摄策略

| 天气 | 受影响的机位 | 替代方案 |
|------|-------------|----------|
| ☀️ **晴天** | 所有露天机位均可 | 注意正午强光（11:00-14:00）避拍 |
| 🌤️ **多云/阴天** | 光线均匀，全天可拍 | 室外人像最佳，肤色均匀 |
| 🌧️ **小雨** | 户外减分，但透明伞可当道具 | 茅家埠/龙井（烟雨江南） |
| ⛈️ **暴雨** | 所有户外机位不适合 | 改期或转室内棚拍 |
| 🌫️ **雾霾/沙尘** | 远景机位减分（龙井远眺/宝石山） | 改拍近景/室内 |
| 🌁 **晨雾** | 云栖竹径/龙井（雾气加成 🔥） | 清晨6-8点限时机位 |

### 各季节拍摄重点

| 季节 | 最佳时段 | 注意事项 |
|------|----------|----------|
| 🌸 **春** | 6:00-17:00 | 花粉过敏、春雨频繁、温差大 |
| ☀️ **夏** | 5:30-18:30 | 高温（>35°C不建议户外）、蚊虫暴增、防晒必需 |
| 🍁 **秋** | 6:30-17:00 | 最佳季节，光线/温度/植被全部理想 |
| ❄️ **冬** | 7:00-16:30 | 天黑早、温度低、户外限时 |

## 五、拍摄日完整规划流程

### 出工前（提前1-3天）

```
1️⃣ 确定拍摄日期、风格、机位
2️⃣ 调用 sunset-prediction 评估天气 ⭐
3️⃣ 查看人流量日历 → 避开高峰
4️⃣ 查看机位细节 → 确认停车/交通
5️⃣ 查看季节植被 → 确认最佳拍摄角度
6️⃣ 发送拍摄前准备清单给客户
```

### 出工当天

```
1️⃣ 查看当天实时天气（建议使用AM/PM两次确认）
2️⃣ 根据天气调整拍摄计划
   ☀️ 晴天 → 黄金时段优先，正午休息/室内
   🌤️ 阴天 → 全天可拍，光线均匀
   🌧️ 小雨 → 备选方案A：透明伞+雨景
   ⛈️ 暴雨 → 启用B计划：改期/室内棚拍
3️⃣ 计算到达时间（预留交通拥堵余量30min）
4️⃣ 到达现场 → 先踩点→再沟通→开拍
```
